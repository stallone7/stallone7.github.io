行政起诉状(王细罗)

发表于 2018-04-11 | 分类于 起诉状

行政起诉状

  • 原告：王细罗

男，汉族，身份证号码：430104196312281517，住址：开福区潮宗街91号，电话:13808452019.

  • 被告一：长沙市开福区人民政府;

    地址:长沙市芙蓉北路开福区政府大院内;邮 编：4100008;联系电话：84558280;

法定代表人：刘拥兵

  • 被告二：长沙市人民政府

地址：中國湖南省长沙市岳麓区岳麓大道218号政府二办公楼0128室;联系电话：88665077

诉讼请求：

  1. 请求确认被告一作出的《房屋征收决定》（开政征补字（2017）95号）违法并予以撤销；

  2. 请求撤销被告二作出的《行政复议决定书》（长府复决字[2018]行政复第55号）；

  3. 本案诉讼费用由两被告承担。

    事实与理由

    2016年5月30日，被告一发布《房屋征收决定》（开政征补字（2017）95号），原告房屋位于该征收决定确定的征收范围内，原告认为被告一作出的房屋征收决定行政行为违法，向被告二长沙市人民政府提起行政复议，被告二于2018年2月7日立案受理，被告二于2018年4月10日作出《行政复议决定书》（长府复决字[2018]行政复第55号），维持被告一作出的《房屋征收决定》。基于以上事实，原告认为：

    1. 征收所涉建设项目未取得建设项目立项批复、建设用地规划许可证、国有土地使用权批准文件以及建设工程规划许可、建设项目环评审批等前置行政行为法律文件。被告一在复议程序中未向原告提供以上前置行政行为合法的法律文件。
    2. 被告一征收行为一说是历史文化街区改造，又说是棚改项目改造，前后矛盾且明显违反《国务院关于加快棚户区改造工作的意见》(国发〔2013〕25号)文件精神禁止将因城市道路拓展、历史街区保护、文物修缮等带来的房屋拆迁改造项目纳入城市棚户区改造范围
    3. 被告一征收行为不符合国民经济和社会发展规划、土地利用整体规划、城乡规划和专项规划及长沙市开福区国民经济和社会发展年度计划；且这些规划的制定未按照《国有土地征收与补偿条例》第9条规定广泛征求社会公众意见，未经科学论证。被告一在复议程序中未向原告提供相关规划和计划的原始证据材料。
    4. 被告一实施征收行为前未依法进行社会稳定风险评估并提供社会稳定风险评估备案呈报表。
    5. 被告一实施征收行为前，未按照《国有土地征收与补偿条例》第12条规定做到征收补偿费用足额到位、专户存储、专款专用；
    6. 征收范围应与建设项目立项批复、建设用地规划许可证、国有土地使用权批准文件确定的范围一致，并且文件出具时间应早于征收决定公告日期；但根据信息公开资料显示：
    • 规划局于2016.08.19才出具选址意见书（在征收决定公告发布日期之后）
    • 国土局于2016.9.1才出具潮宗街棚改一期预审意见（在征收决定公告发布日期之后）
    1. 被告一未依法作出《房屋征收决定书》并按照法定方式公告。
    2. 被告一未将补偿方案征求意见情况和根据公众意见修改情况进行及时公布。被告一亦未组织针对补偿方案的听证会并根据听证会情况修改补偿方案。
    3. 征收补偿方案中关于评估机构的确定，违反《国有土地上房屋征收评估办法》第四条规定房地产价格评估机构由被征收人在规定时间内协商选定；在规定时间内协商不成的，由房屋征收部门通过组织被征收人按照少数服从多数的原则投票决定，或者采取摇号、抽签等随机方式确定。被告一在复议程序中也未向被告二提供证明被征收人在规定时间内协商选定的原始证据材料。
    4. 征收补偿方案未提供改建地段或就近地段的拥有合法产权、质量安全的房屋让原告等被征收人选择产权调换房屋，违反《国有土地征收与补偿条例》第21条第二款的规定因旧城区改建征收个人住宅，被征收人选择在改建地段进行房屋产权调换的，作出房屋征收决定的市、县级人民政府应当提供改建地段或者就近地段的房屋。被告一的行为损害了原告的选择权。

综上，被告一作出的房屋征收决定事实认定不清，适用法律错误，程序违法；征收补偿方案未能充分保障原告等被征收人的合法权利。被告二复议决定认定事实不清，适用法律错误，程序违法。故原告为了维护自身的合法权益，特诉至贵院，希判如所请。

此致

长沙市中级人民法院

具状人：王细罗

2018年4月11日
